**Response:**
```json

  "message": "FindYourPeace API is running",
```

#### `GET /api/feelings`
Retrieve all emotional states.

**Response:**
```json
[
  {
    "id": "happy",
    "label": "Happy / Grateful",
    "description": "Feeling joyful and thankful",
    "verse_count": 10
  }
]
```

#### `POST /api/verse`
Get a personalized verse based on emotional state.

**Request Body:**
```json
{
  "feeling_id": "anxious",
  "user_id": "user123",
  "exclude_verses": ["2:255", "94:5"]
}
```
